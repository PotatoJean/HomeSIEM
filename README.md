<h1>Home SIEM Lab</h1>

<h2>Description</h2>

This is my attempt to create a home lab running SIEM with Elastic and Kali Linux Virtual Machine to familiarize myself with the creation of alerts from logs.</br>
</br>

- <b>Elastic Web Interface</b>

<h2>Conclusion</h2>
<p>To sum everything up, we have done the following: 

-set up home lab using elastic SIEM and kali vm
-forwarded data from kali to SIEM using elastic beats agent
-generated security events on kali vm using Nmap
-queried and analysed logs in SIEM using elastic web interface
-created dashboard to visualise security events
-created an alert to detect security events
